# Installation

## Requirements

- Go 1.21 or higher
- A supported database (PostgreSQL, MySQL, or SQLite)

## Install lit

```bash
go get github.com/tracewayapp/lit
```

## Database Drivers

lit works with standard `database/sql` drivers. Install the driver for your database:

### PostgreSQL

```bash
# pgx (recommended)
go get github.com/jackc/pgx/v5/stdlib

# or pq
go get github.com/lib/pq
```

### MySQL

```bash
go get github.com/go-sql-driver/mysql
```

### SQLite

```bash
go get github.com/mattn/go-sqlite3
```

You can also use a [custom driver](/guides/custom-drivers) to support databases not listed above.

## Verify Installation

Create a simple program to verify everything is working:

```go
package main

import (
    "fmt"
    "github.com/tracewayapp/lit"
)

type User struct {
    Id    int
    Name  string
    Email string
}

func main() {
    lit.RegisterModel[User](lit.PostgreSQL) // or lit.MySQL, lit.SQLite
    fmt.Println("lit installed successfully!")
}
```

Run it:

```bash
go run main.go
```

If you see "lit installed successfully!", you're ready to proceed to the [Quick Start](/getting-started/quick-start).
